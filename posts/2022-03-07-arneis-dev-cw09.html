Released code to pretty-print LEGO BOM

You may have noticed that the ARNEIS documentation site contains a detailed list of all the LEGO® pieces included in the Set 42100 which we have selected to build our MOC.

You may wonder how we managed to create this page.

The secret is lego-parts-csv2md, a Jupyter notebook developed by Alessandro which is able to parse the Bill Of Material exported from Stud.io CAD tool and convert it to a snippet of Markdown code.

Since we believe this tool may be useful to other people as well, we decided to publish the source code under the Open Source MIT license, like all the other contents of the ARNEIS repository on GitHub.

You may find the source code of lego-parts-csv2md, included its usage instructions, under the /code folder of the GitHub repository.
